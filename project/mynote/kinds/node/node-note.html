1.nodejs是什么

后端语言: 它是通过js代码编写，调用的api封装是好了的底层C++代码，并不是通过js代码转换成了C++；

nodejs是对是对Google V8引擎进行了封装。V8引 擎执行Javascript的速度非常快，性能非常好。Node.js对一些特殊用例进行了优化，提供了替代的API，使得V8在非浏览器环境下运行得更好。

异步编程：执行一个命令时不会影响后面的程序执行，大量的异步编程使效率大大提升。

2. 终端常用的命令

  • cd 进入目录 （按tab可以自动补全目录）
  • 换盘直接 盘符: （比如 D:）
  • cls 清空当前终端的内容
  • （powsershell） clear 快捷键ctrl+L 清空当前终端的内容
  • dir 查看当前目录中的内容
  • （powsershell） ls 查看当前目录中的内容
  • 方向键上下查看历史命令
  • history：查看所有历史敲过的命令

fs模块

x
nodejs所有异步操作api都必须有回调函数，并且回调函数的第一个形参是错误对象(没有错误是null，有错误是对象)
fs.readFile(文件路径，文件格式，回调函数);

  1. 读文件：fs.readFile(文件路径，文件格式，回调函数);

  2. 写文件：writeFile(参数1，参数2，参数3，参数4) 参数1：路径 参数2：内容 参数3：options选项 参数4：回调

    writeFile：不能创建一个不存在的文件夹

  3. options中常用的参数 1、encoding：编码格式 (可以省略) 2、flag: 标志 默认是w(覆盖) a(追加)

  4. 同步API : 原有的名称+Sync,没有回调函数

  5. unlink(参数1，参数2) ： 删除文件(不能删除目录，文件不存在会报错) 参数1：文件路径 参数2：回调函数

  6. readdir(参数1，参数2) : 读目录（只会读取一层） 参数1：路径 参数2：回调函数

  7. mkdir(参数1，参数2) : 创建目录 参数1：路径 参数2: 回调函数

  8. rmdir(参数1，参数2) : 删除文件夹（不能删除非空文件夹） 参数1：路径 参数2: 回调函数

案例：删除该路径的所有文件(系统占用的无法删除)

​
const fs = require('fs');
const path = require('path');
​
let rm = (PAT)=>{
    try {
        let curStat = fs.statSync(PAT);
​
        if(curStat.isFile()){
            fs.unlinkSync(PAT);
        }
​
        if(curStat.isDirectory()){
            let curDir = fs.readdirSync(PAT);
​
            if(curDir.length){
                curDir.forEach(item=>{
                    let NEW_PATH = path.join(PAT,item);
                    rm(NEW_PATH);
                })
            }
            fs.rmdirSync(PAT);
        }
    }catch (e) {
        console.log(e);
    }
};
​
module.exports = rm;
