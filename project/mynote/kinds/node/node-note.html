1.nodejs是什么

后端语言: 它是通过js代码编写，调用的api封装是好了的底层C++代码，并不是通过js代码转换成了C++；

nodejs是对是对Google V8引擎进行了封装。V8引 擎执行Javascript的速度非常快，性能非常好。Node.js对一些特殊用例进行了优化，提供了替代的API，使得V8在非浏览器环境下运行得更好。

异步编程：执行一个命令时不会影响后面的程序执行，大量的异步编程使效率大大提升。

2. 终端常用的命令

  • cd 进入目录 （按tab可以自动补全目录）
  • 换盘直接 盘符: （比如 D:）
  • cls 清空当前终端的内容
  • （powsershell） clear 快捷键ctrl+L 清空当前终端的内容
  • dir 查看当前目录中的内容
  • （powsershell） ls 查看当前目录中的内容
  • 方向键上下查看历史命令
  • history：查看所有历史敲过的命令
