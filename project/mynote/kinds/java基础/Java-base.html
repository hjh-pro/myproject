1.1计算机发展史

第一代计算机：（40年代中期到50年代末期） 电子管计算机，体积庞大，运算速度慢，功耗高，可靠性差，价格昂贵。应用领域军事和科学。

第二代计算机：（50年代末期到60年代中期） 晶体管计算机，体积缩小，运算速度提高，功耗降低，可靠性提高，运算速度提高，性能比第一代大大提高。开始向非数值方向发展，应用的领域主要是科学计算、企业信息管理。

第三代计算机：（60年代中期到70年代初期） 小规模集成电路计算机，运算速度显著提高，可靠性提高，价格降低。应用领域主要是文字处理，图像图形处理，基本上进入到经济、工业、科学等各个领域。

第四代计算机：（70年代至今） 大规模集成电路计算机新时代，硬件更加精化，软件更加丰富，走向家庭应用，面向社会。

2.4java基本数据类型的转换

自动转换

数据范围：

转换规则：

1.所有的byte类型和short类型和char值运算的时候都会提供成int类型，int最大值2147483647。

2.如果计算中一个操作数据类型是long，另一个数的空间小于等于long，那么结果就是long

3.如果计算中一个操作数据类型是float，另一个数的空间小于等于float，那么结果就是float

4.如果计算中一个操作数据类型是double，另一个数的空间小于等于double，那么结果就是double

*注意1：int long float, double 自身来计算的时候没有类型的提升，如果超出范围了就会出现精度的损失

2：int long float, double 不同的类型来计算的时候会向大的表数范围提升

        float f1 = (float)d;

        char c = 'a';
        int x = c;
        System.out.println(x); // 97
        
        int y = 98;
        char c1 = (char)y;

    }
}

数据类型[] 数组名 = new 数据类型[整数];

画图理解：

基本数据类型的变量都存储在栈中，栈的特点是存储空间小，但是存取速度快，先进后出。

引用数据类型所占的空间比较大，存储在堆中，堆的特点，空间大，存取的速度比较慢。(相对栈存取较慢)

整数类型（byte, short, int, long）的默认值都是 0

小数(float, double)的默认值都是 0.0

布尔数据类型(boolean)的默认值是 false

字符数据类型(char)的默认值是 ‘ ’ (空格)

练习一：数组求最大值

class Demo15{
    public static void main(String[] args){
        //定义一个整数类型的数组
        int[] arr = {12, 4, 5, 78, 94, 24};
        
        int result = max(arr);
        
        System.out.println("数组的最大值是："+result);
        
    }
    /*
        数组是引用数据类型，引用数据类型有可能发生空指向的问题
        //引用数据类型可以赋值成null，这就代表是空指向
        int[] arr1 = null;
    
    */
    
    public static int max(int[] arr){
        //定义一个最大值的结果变量
        int maxValue = -1;
        //我们为了程序的严谨性，一定要判断null和0长度的问题
        if(arr != null && arr.length != 0){
            for(int i = 0; i < arr.length; i++){
                //如果遍历的数组中的值比最大值变量大的话，需要更换
                if(arr[i] > maxValue){
                    maxValue = arr[i];
                }
            }
            
        }
        
        return maxValue;
    }
    
}

练习二：冒泡排序(将数组从大到小排序)

class Demo16{
    public static void main(String[] args){
        //定义一个整数类型的数组
        int[] arr = {23, 25, 12, 7, 51, 34,66,789};
        
        sortPop(arr);
        for(int i = 0; i < arr.length; i++){
            System.out.print(arr[i]+"\t");
        }
        
    }
    
    public static void sortPop(int[] arr){
        if(arr != null && arr.length != 0){
            //外层循环是轮数
            for(int i = 0; i < arr.length - 1; i++){
                //内层循环是每一轮比较的次数
                for(int j = 0; j < arr.length - i - 1; j++){
                        if(arr[j+1] > arr[j]){
                            //定义一个临时的变量
                            int temp = arr[j];
                            arr[j] =  arr[j+1];
                            arr[j + 1] = temp;
                        }
                }
            }
        }
        
    }
}

class Demo17{
    public static void main(String[] args){
        //定义一个整数类型的数组
        int[] arr = {23, 25, 12, 7, 51, 34, 66, 789};
        reverseArr(arr);
        
        for(int i = 0;i < arr.length; i++){
            System.out.print(arr[i]+"\t");
        }
        
    }
    
    public static void reverseArr(int[] arr){
        
        for(int i = 0; i < arr.length/2; i++){
            //交换前面和后面的对称的值
            int temp = arr[arr.length - 1- i];
            arr[arr.length - 1- i] = arr[i];
            arr[i] = temp;  
        }
    }
}
